## Lerna Usage

### Running tests

```bash
$ npm run lerna:test
```

### Bootstrapping the packages

```bash
$ npm run lerna:bootstrap
```

### Installing a new package

```bash
$ npm run lerna:install -- <package-name> --scope=<target-package>
```
